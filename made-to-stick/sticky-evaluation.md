## Quick Fixes by Principle

### If Simple is weak:
- [ ] Cut the idea to one sentence
- [ ] Use the "Commander's Intent" format
- [ ] Remove all secondary points

### If Unexpected is weak:
- [ ] Add a pattern break in the opening
- [ ] Create a curiosity gap (question, mystery)
- [ ] Find what's counterintuitive

### If Concrete is weak:
- [ ] Replace every abstraction with an example
- [ ] Add sensory details
- [ ] Include specific numbers

### If Credible is weak:
- [ ] Add your Sinatra Test
- [ ] Make statistics human-scale
- [ ] Let them verify the claim themselves

### If Emotional is weak:
- [ ] Focus on one individual, not statistics
- [ ] Connect to their identity
- [ ] Add WIIFY (What's In It For You)

### If Stories is weak:
- [ ] Add a protagonist with a challenge
- [ ] Use before/after narrative
- [ ] Make it possible for them to see themselves

## Curse of Knowledge Check

For technical or expert content, additionally verify:

- [ ] All acronyms defined on first use
- [ ] No assumed background knowledge
- [ ] Every concept has a concrete example
- [ ] Analogies to familiar things included
- [ ] A non-expert reviewed and understood it
