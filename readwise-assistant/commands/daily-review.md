# Daily Review

Get your daily spaced repetition review highlights using the `get_daily_review` MCP tool.

## Usage

```
/readwise:daily-review
```

## Instructions

Call the `get_daily_review` tool (no arguments needed).

Present the review highlights in a clear format with:
- Review ID and URL
- Number of highlights
- Each highlight with book context
- Whether review is completed

Explain that these are highlights selected by Readwise's spaced repetition algorithm for optimal retention.

## About Spaced Repetition

Readwise uses spaced repetition to surface your highlights at optimal intervals for long-term retention. The daily review presents highlights you haven't seen in a while, helping you remember what you've read.

## Example

```
/readwise:daily-review
```
