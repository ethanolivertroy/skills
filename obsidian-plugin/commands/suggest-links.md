Use AI to suggest links between notes based on content similarity.

Usage:
  /obsidian:suggest-links <note-name>
  /obsidian:suggest-links --all

Examples:
  /obsidian:suggest-links my-note
  /obsidian:suggest-links --all

This command:
- Analyzes note content using Claude
- Finds semantically related notes in your vault
- Suggests bidirectional links to add
- Shows confidence scores for each suggestion
